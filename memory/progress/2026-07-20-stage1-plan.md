  (stereo cam0/cam1 @ 20Hz, IMU @ 200Hz, Leica + Vicon-fused ground truth,

- Wrote `plan/STAGE1.md`: an 11-milestone (M0-M10) plan to build a

  work needs `cargo test` coverage *and* an update to a human-readable test
  app (`bin/slam-inspect`, not yet created) so progress is checkable without
  reading code; this `memory/` directory for cross-session continuity;
  commit-and-push-to-origin after each meaningful unit of progress
  (pre-authorized, no need to ask each time).
